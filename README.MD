## 🛠 Tech Stack

This portfolio was built using a modern frontend stack to ensure speed, responsiveness, and cinematic visuals:

* **Frontend:** HTML5, CSS3 (Custom Glassmorphism UI)
* **Framework:** [Bootstrap 5.3](https://getbootstrap.com/)
* **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/), [Animate.css](https://animate.style/)
* **Interactivity:** JavaScript (ES6+)
* **Contact System:** [EmailJS](https://www.emailjs.com/) (Direct Client-to-Email integration)
* **Icons:** Font Awesome 6

---

## ✨ Key Features

- **Dark Tech Theme:** A high-end visual identity focused on AI and futuristic aesthetics.
- **Glassmorphism Design:** Modern translucent UI elements with blur effects.
- **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).
- **Interactive Portfolio:** Dynamic project showcase with hover animations.
- **Automated Contact Form:** Functional contact page powered by EmailJS API.
- **Smooth Animations:** Integrated AOS for a premium scrolling experience.
